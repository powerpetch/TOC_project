About This Project

A Theory of Computation project applying Regular Expressions to web scraping and data extraction.

Overview

This project demonstrates the practical application of Regular Expressions — a core concept in the Theory of Computation — by building a web scraper that extracts data about professional Valorant players from Liquipedia.

Regular expressions are equivalent to Finite Automata (DFA/NFA) in computational theory. Every regex pattern used in this project can be represented as a DFA state machine that accepts or rejects input strings.

Features

  • Web scraping from Liquipedia using 12 regex patterns with DFA/NFA theory
  • 149 professional Valorant player profiles with detailed information
  • Searchable, sortable, and filterable player table (by role, country, team)
  • Detailed player profile pages with career stats and social links
  • Analytics dashboard with interactive Chart.js visualizations
  • Interactive regex tester — test patterns live with step-by-step match traces
  • Visual DFA state machine diagrams for each regex pattern
  • Regex patterns page with full DFA/NFA explanations
  • REST API endpoints for player data (/api/players, /api/stats)
  • Responsive dark theme design

Data Source

All player data is scraped from Liquipedia Valorant. The scraper visits team pages to collect player information including IGN, real name, nationality, role, signature agents, earnings, and social media links. All extraction uses Python's re module.
